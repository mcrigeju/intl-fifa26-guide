# intl-fifa26-guide

本仓库用于归档和发布多个独立的 HTML 页面。这些页面内容彼此独立，不针对任何特定域名或网站。

## 项目简介

intl-fifa26-guide 是一个静态 HTML 页面集合仓库。你可以在此找到一系列独立开发的页面文件，每个页面均以完整 HTML 文档形式存在，可直接在浏览器中打开查看。

## 目录说明

```
intl-fifa26-guide/
├── docs/               # 归档的 HTML 页面存放目录
│   ├── page-001.html   # 示例页面一
│   ├── page-002.html   # 示例页面二
│   └── ...
├── assets/             # 共用的资源文件（图片、样式、脚本等）
└── README.md           # 本说明文件
```

- `docs/`：主要归档目录，所有独立 HTML 页面均存放于此。
- `assets/`：存放页面可能引用的共用资源，如图片、CSS 或 JavaScript 文件。

## 页面归档说明

- 每个 HTML 页面均为独立文档，包含完整的 `<html>`、`<head>` 和 `<body>` 结构。
- 页面之间没有依赖关系，可单独打开使用。
- 归档的页面内容可能涉及多种主题或用途，请根据实际需要浏览。
- 页面文件命名采用 `page-xxx.html` 格式，便于识别和排序。

## 使用方式

1. 克隆本仓库到本地：
   ```bash
   git clone https://github.com/your-username/intl-fifa26-guide.git
   ```
2. 进入 `docs/` 目录，使用浏览器直接打开任意 `.html` 文件即可查看。
3. 也可通过 GitHub Pages 等静态托管服务发布 `docs/` 目录内容（需自行配置）。

## 维护说明

- 本仓库主要进行页面新增、更新或清理维护。
- 新增页面请放入 `docs/` 目录，并保持命名规范。
- 如有共用资源更新，请放入 `assets/` 目录，并确保页面引用路径正确。
- 不接收针对特定域名或网站的页面提交。
- 欢迎提交 Issue 或 Pull Request 以协助改进页面内容或结构。
